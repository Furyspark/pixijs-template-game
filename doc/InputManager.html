InputManager

Handles user input.

Members

<static, readonly> buttons :Object.<Input_Key>

A dictionary for keyboard and mouse buttons.
Type:
  • Object.<Input_Key>

<static, readonly> gamepads :Array.<Input_Gamepad>

Contains 16 gamepads, not all of which may be connected.
Type:
  • Array.<Input_Gamepad>

<static, readonly> position :Object

An object with certain input positions. Contains a variable 'mouse', which contains variables 'screen' and 'game', which are both points.
Type:
  • Object
Examples
sprite.x = InputManager.position.mouse.game.x
sprite.y = InputManager.position.mouse.game.x
// Returns whether the mouse position is in the top-left corner of the screen.
if(InputManager.position.mouse.screen.x === 0 && InputManager.position.mouse.screen.y === 0) {
  this.doSomething();
}
